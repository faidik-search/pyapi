Faidik Search API

Документация для разработчиков

Введение

Faidik Search API позволяет получать информацию о номерах телефонов. API возвращает данные в формате JSON.

Получение доступа

Для доступа к API необходим токен, который можно приобрести у @W2SP3R в Telegram.

Токен предоставляет доступ ко всем возможностям API и необходим для авторизации запросов.

Связаться с @W2SP3R

Базовый URL

GET https://faidik-search.github.io/pyapi/

Эндпоинты

Поиск по номеру телефона

GET Поиск информации по номеру телефона
https://faidik-search.github.io/pyapi/?number={phone}&token={your_token}

Пример запроса

GET https://faidik-search.github.io/pyapi/?number=79001234567&token=ВАШ_ТОКЕН

Пример ответа

{
  "results": [
    {
      "fio": "Гаянов Р Г",
      "address": "ХМАО НИЖНЕВАРТОВСК, НАБЕРЕЖНАЯ, 5 КВ.35",
      "phone": 79044601473,
      "data": "himera_2023"
    }
  ],
  "search_time_seconds": 0.691
}

Соотношение переменных

Поле Тип Описание
fio String ФИО владельца номера телефона
address String Адрес регистрации владельца
phone Number Номер телефона в формате 79XXXXXXXXX
data String Источник данных
search_time_seconds Float Время выполнения запроса в секундах

Примеры кода

Python

# Установите requests: pip install requests
import requests

# Базовый URL API
base_url = "https://faidik-search.github.io/pyapi/"
phone_number = "79001234567"
api_token = "ВАШ_ТОКЕН" # Замените на ваш токен

# Формируем запрос
params = {
  "number": phone_number,
  "token": api_token
}

# Отправляем запрос
response = requests.get(base_url, params=params)
data = response.json()
print(data)

JavaScript

// Базовый URL API
const baseUrl = 'https://faidik-search.github.io/pyapi/';
const phoneNumber = '79001234567';
const apiToken = 'ВАШ_ТОКЕН'; // Замените на ваш токен

// Формируем URL запроса
const url = `${baseUrl}?number=${phoneNumber}&token=${apiToken}`;

// Используйте fetch API
fetch(url)
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

cURL

# Базовый URL API
BASE_URL="https://faidik-search.github.io/pyapi/"
PHONE="79001234567"
TOKEN="ВАШ_ТОКЕН" # Замените на ваш токен

curl "$BASE_URL?number=$PHONE&token=$TOKEN"

Важно

Замените ВАШ_ТОКЕН на приобретенный токен. Без valid токена запросы не будут работать.
